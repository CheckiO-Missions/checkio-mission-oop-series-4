Ця серія завдань допоможе вам ознайомитись з основами ООП у Python, такими як створення класів, атрибутів та методів, наслідування, перевизначення методів, використання super() та поліморфізм.

Місія 4: Методи класу.

  • 4.1 Додай властивість об'єкта working_engine з початковим значенням False і зміни метод __init__ відповідним чином.
  • 4.2 Додай метод без позиційних аргументів start_engine до класу Car, який виводить повідомлення "Engine is started" і змінює значення working_engine на True.
  • 4.3 Додай метод без позиційних аргументів stop_engine до класу Car, який виводить повідомлення "Engine is stopped" і змінює значення working_engine на False.
  • 4.4 Виклич метод start_engine для обох об'єктів з попередньої місії.
