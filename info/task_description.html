This series of tasks will help you learn the basics of OOP in Python, such as creating classes, attributes, and methods, inheritance, method overrides, using super(), and polymorphism.

Mission 4: Class methods.

4.1. Add an object property working_engine inside __init__ method of Car class. Set its default value to False. Do NOT change arguments, which are passing into the method.

4.2. Add a start_engine method to the Car class, that displays the message "Engine has started" and changes the working_engine value to True.

4.3. Add a stop_engine method to the Car class, that displays the message "Engine has stopped" and changes the working_engine value to False.

4.4. Call the start_engine method for both some_car1, some_car2.
